Files in this folder provide human interaction data for the curator tool llm app. Currently human interactions downloaded from IntAct and BioGrid are used due to their PubMed associations.

BioGrid: https://downloads.thebiogrid.org/File/BioGRID/Release-Archive/BIOGRID-4.4.243/BIOGRID-ORGANISM-4.4.243.tab3.zip. All species files are in this zipped file. However, only human PPIs file is kept.

IntAct: https://www.ebi.ac.uk/intact/interactomes, miTab 2.7 is downloaded. The downloaded file name is human.txt. IntAct_ was added afterwards.
